Semestre V

El Quinto Semestre marcó un salto hacia la Arquitectura Empresarial y la Gestión de Calidad.

Objetivos Específicos

  • Desarrollar una API REST funcional que permita la comunicación eficiente entre la base de datos y la aplicación Boss Budget.
  • Implementar una herramienta interna de gestión ISO 9001 y 27001 orientada al control de procesos de calidad y seguridad.
  • Realizar pruebas de calidad exhaustivas que aseguren la conformidad de la aplicación con los estándares de desarrollo.
  • Diseñar y desplegar un Dashboard interactivo en Power BI para visualizar métricas clave del sistema.
  • Optimizar la arquitectura empresarial del proyecto bajo un enfoque modular y sostenible.
  • Fomentar la colaboración en equipo aplicando metodologías ágiles (Scrum).

API REST: El Corazón de la Conexión

Para asegurar que BossBudget pudiera comunicarse eficientemente con nuestra base de datos y futuros servicios, desarrollamos una robusta API RESTful.

  • Funcionalidad Clave: Manejo de usuarios, transacciones y presupuestos.
  • Tecnologías:Utilizamos un stack moderno para garantizar escalabilidad y alto rendimiento.
  • Arquitectura: Definición clara de endpoints (GET, POST, PUT, DELETE) siguiendo los principios REST.

Herramienta de Gestión de Calidad y Seguridad (ISO)

Software para ISO 9001 y 27001

Desarrollamos una herramienta interna para facilitar la gestión de la calidad (ISO 9001) y la seguridad de la información (ISO 27001) del proyecto.

  • ISO 9001: Módulos para control de procesos y documentación.
  • ISO 27001: Registro de riesgos, gestión de activos y políticas de seguridad.
  • Objetivo: Garantizar la confiabilidad y legalidad de BossBudget.

Documento de Prueba de Calidad

Se realizó una rigurosa Prueba de Calidad que evaluó la herramienta contra los requisitos específicos de ambas normas ISO para certificar su cumplimiento.
