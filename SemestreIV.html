SEMESTRE IV

Conocimientos y tecnologías que impulsaron el avance de la aplicación Boss Budget durante este periodo.

Tecnologías Abordadas

Objetivos Específicos

  • Fortalecer la gestión y administración de bases de datos mediante la creación de usuarios y asignación de permisos en MySQL Workbench, garantizando la seguridad y correcta distribución de roles dentro del equipo de desarrollo.
  • Implementar técnicas de encriptación de datos sensibles en la base de datos del proyecto Boss Budget para reforzar la protección y confidencialidad de la información de los usuarios.
  • Diseñar el maquetado de la aplicación.
  • Desarrollar y optimizar interfaces web utilizando HTML, Tailwind CSS y JavaScript, aplicando buenas prácticas de diseño responsivo y usabilidad.
  • Diseñar hojas de vida digitales interactivas como ejercicio práctico de maquetación en HTML, CSS y JS, fortaleciendo la presentación profesional de los integrantes del equipo.
  • Crear animaciones 2D aplicadas al proyecto Boss Budget empleando herramientas como Renderforest, Animaker y similares, para mejorar la comunicación visual y la creatividad en el desarrollo.
  • Gestionar los recursos del proyecto en un repositorio organizado en GitHub, centralizando el código, documentación y material multimedia del portafolio académico.
  • Aplicar herramientas complementarias como Excel, Access y Power BI para el análisis y la visualización de datos financieros vinculados al funcionamiento del proyecto.

Introducción al Cuarto Semestre

Para empezar el cuarto semestre decidimos crear un repositorio directamente para lo relacionado al proyecto, para tener un mejor orden y presentación del portafolio y página web, aparte de almacenar todo el código de la aplicación Boss Budget.

Creación de Usuarios y Asignación de Permisos (MySQL)

Para el manejo de una base de datos entre varios integrantes es necesario que cada uno tenga previamente asignado un usuario y unos permisos correspondientes a su cargo. Para ello, se profundizó en el cómo crear usuarios y asignar permisos en MySQL Workbench, esto se realizó en la Gestión y Administración de Bases de Datos.

Encriptación de Datos en MySQL

La protección de las bases de datos es un factor fundamental a tener en cuenta a la hora de crear un proyecto. Por ende, es necesario llevar a cabo una encriptación de los datos sensibles. Esto fue parte de lo que vimos en el segundo corte del cuarto semestre en la clase de Gestión y Administración de Bases de Datos.
