¡Bienvenidos a nuestro Portafolio del Cuarto Semestre!

En esta sección podrás conocer una breve introducción a los conocimientos y tecnologías que abordamos durante el semestre, así como una recopilación visual de las herramientas que manejamos para los distintos proyectos. En este cuarto semestre vimos algunas herramientas y funcionalidades que nos ayudaron para avanzar en la creación del aplicativo, como la edición de videos, el manejo de excel, la creación de usuarios en mysql y más.

Introducción al cuarto semestre

Para empezar el cuarto semestre decidimos crear un repositorio directamente para lo relacionado al proyecto, para tener un mejor orden y presentación del portafolio y página web, aparte de almacenar todo el código de la aplicación BossBudget.

Creación de Usuarios de una Base de Datos

Para el manejo de una base de datos entre varios integrantes es necesario que cada uno tenga previamente asignado un usuario y unos permisos correspondientes a su cargo en el desarrollo, para ello vimos cómo crear usuarios y asignar permisos en MySQL Workbench, todo en la clase de Gestión y Administración de Bases de Datos.

Hojas de Vida de los Integrantes en HTML

Para la clase de técnicas de animación vimos los párametros para diseñar una hoja de vida en HTML, CSS y JS. Conoce a los Desarrolladores de BossBudget haciendo clic en el botón debajo de cada imagen para ver su hoja de vida.

Encriptación de Datos en MySQL

La protección de las bases de datos es un factor fundamental a tener en cuenta a la hora de crear un proyecto, por ende es necesario llevar a cabo una encriptación de los datos sensibles, eso fue parte de lo que vimos en el segundo corte del cuarto semestre en la clase de Gestión y Administración de Bases de Datos.
